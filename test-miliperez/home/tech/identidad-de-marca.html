03

Valores Core

5 principios no negociables que guían cada proyecto, cada email, cada decisión. No están aquí para sonar bien. Traicionarlos sería traicionar la marca.

  • 01

    Honestidad Brutal

    Si algo no funciona, se dice. Si un proyecto no es viable, se dice. Si hay una forma mejor, se dice. Sin reuniones para justificar horas. Sin informes para parecer ocupado. Tiempo es dinero. El tuyo y el mío.

  • 02

    Autonomía Radical

    No trabajo en oficinas. No hago reuniones si un email resuelve el tema. No explico dónde estoy. Trabajo a las 3AM si ahí llegan las mejores ideas. Lo que importa no es cuándo o dónde, sino que el sistema funcione.

  • 03

    Resultados sobre Procesos

    No me importa cómo se haga mientras funcione. No sigo metodologías porque están de moda. Si algo funciona, se queda. Si no, se tira. El resultado es lo único que cuenta.

  • 04

    Autenticidad sin Postureo

    Cero teatro. Cero frases motivacionales vacías. Cero fotos en coworking con Mac y café. El trabajo habla. Los resultados gritan. Todo lo demás es ruido.

  • 05

    Transformación Real

    No vendo consultorías para tener consultorías eternas. Te enseño a pescar. Mi objetivo es hacerme innecesaria. Si tu equipo vuela solo después de trabajar conmigo, he hecho bien mi trabajo.

06

Tono de Voz

Cómo suena Mili Pérez Tech cuando habla. El cómo es tan importante como el qué. Cada palabra cuenta. Cada frase define.

Lo que SÍ hacemos

  • Frases cortas. Puntos. Sin relleno.
  • Tú/Tus. Cercanía con autoridad.
  • Verbos de acción en presente: "Construyo", "Resuelvo", "Entrego"
  • Datos concretos: "14 años", "50% de anticipo", "48h de respuesta"
  • Verdades incómodas dichas con gracia
  • Preguntas directas que obligan a pensar
  • "Si X, entonces Y" - lógica clara

Lo que NO hacemos

  • "Explorar infinitas posibilidades de transformación digital..."
  • Frases subordinadas kilométricas que no dicen nada
  • Corporativismo vacío: "sinergias", "stakeholders", "KPIs"
  • Promesas genéricas sin números
  • Motivación barata de LinkedIn
  • "Querido cliente" o "Estimado señor"
  • Eufemismos para suavizar malas noticias
Ejemplo: Inicio de email
"Hola, David. Vi tu formulario. Tres cosas antes de seguir..."
Ejemplo: Rechazo de proyecto
"No puedo ayudarte con esto. No es mi especialidad y te haría perder tiempo. Te recomiendo hablar con X, que sí lo hace bien."
Ejemplo: Propuesta de valor
"Identifico dónde pierdes dinero. Construyo la solución. Te entrego las llaves. Tú solo cobras."

Solo para elementos técnicos: códigos, números de versión, datos técnicos, referencias a herramientas.

Reglas tipográficas

Inter 100-300 para títulos grandes. Inter 400-500 para texto. JetBrains Mono solo para código. Letter-spacing amplio en mayúsculas (0.2em mínimo). Line-height 1.6-1.8 para legibilidad. Nunca usar Comic Sans, Papyrus, o fuentes manuscritas. Nunca.

12

Guía de
Aplicación Visual

Cómo aplicar la identidad en cada contexto. Reglas claras, resultados consistentes.

ESPACIADO Y MÁRGENES

Sistema 8px

Todo se basa en múltiplos de 8: 8, 16, 24, 32, 40, 48, 56, 64, 80px. Padding mínimo: 24px. Separación entre elementos: 40px. Margen de respiración: 80px en desktop, 40px en mobile.

JERARQUÍA TIPOGRÁFICA

5 niveles máximo

H1: 64px / weight 100 / line 1
H2: 40px / weight 200 / line 1.1
H3: 24px / weight 300 / line 1.2
Body: 16px / weight 400 / line 1.6
Small: 12px / weight 500 / line 1.4

USO DE COLOR

80/15/5 Rule

80% Negro (fondos, espacios)
15% Blanco/Grises (texto, elementos)
5% Azul (CTAs, acentos, focus)
Nunca más de 3 colores en una composición.

IMÁGENES Y GRÁFICOS

Alto contraste, sin personas

Fotografía: B/N con tinte azul sutil. Contraste alto. Gráficos: Líneas, no rellenos. Geometría simple. Nunca stock photos. Nunca fotos de equipo. Si necesitas humanos, usa siluetas abstractas.

COMPOSICIÓN

Grid de 12 columnas con mucho aire

Desktop: 12 columnas, gutter 32px. Contenido máximo 1200px centrado. Tablet: 8 columnas, gutter 24px. Mobile: 4 columnas, gutter 16px. Nunca llenar todo el espacio. El vacío es poder. El negro es espacio.
